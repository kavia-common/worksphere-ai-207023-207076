# WorkSphere AI — Web Frontend (Next.js)

Baseline Next.js frontend for WorkSphere AI.

## Requirements
- Node.js 18+ (recommended)

## Getting Started
Install dependencies:
```bash
npm install
```

Run the dev server:
```bash
npm run dev
```

Open:
- http://localhost:3000

## Scripts
- `npm run dev` — start development server
- `npm run build` — build for production
- `npm start` — start production server
- `npm run lint` — run eslint
